O que é HTML?

HTML, ou HyperText Markup Language, é a linguagem padrão usada para criar páginas da web. Ele estrutura o conteúdo da web e é composto por uma série de elementos que dizem ao navegador como exibir o conteúdo

Estrutura:

Um documento HTML básico começa com(!DOCTYPE html), seguido por (html), (head), e (body). O conteúdo visível da página é colocado dentro do (body).

Elementos:

HTML usa "tags" para criar elementos. Por exemplo, (h1) é uma tag que cria um título, enquanto (p) cria um parágrafo.
